# brainstorms/

Decisions-pre-planning. Output of the `brainstorming` skill.

One file per topic: `brainstorms/<topic>.md`. Theme-parallel projects may
use `brainstorms/<theme>/<topic>.md`. Gitignored — durable enough to
reference within a session, transient enough not to ship in the repo.
Methodology calls that should outlive the brainstorm graduate to
`decisions/YYYY-MM-DD_<slug>.md`.

Format and handoff to the planning skill: `.claude/conventions/brainstorm-format.md`.
Rationale: `docs/brainstorm-mechanism.md`.
